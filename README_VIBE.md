## The Feature

Adds text zoom to Dino XMPP client:
- `Ctrl++` - Zoom in (increase font by 10%)
- `Ctrl+-` - Zoom out (decrease font by 10%)
- `Ctrl+0` - Reset to default (100%)

Font size persists across restarts and only affects Dino (not system-wide).

## Why This Exists

The official Dino project has [issue #978](https://github.com/dino/dino/issues/978) 
requesting this feature since 2021, but it hasn't been implemented. This fork 
demonstrates what AI can do when asked to implement a feature.

## Building

### Install Dependencies (Debian/Ubuntu)

```bash
sudo apt-get install -y meson ninja-build valac \
    libgtk-4-dev libadwaita-1-dev libgee-0.8-dev libgpgme-dev libgcrypt20-dev \
    libsqlite3-dev libqrencode-dev libnice-dev libomemo-c-dev libsrtp2-dev \
    libsoup-3.0-dev libwebrtc-audio-processing-dev gettext libgstreamer1.0-dev \
    libgstreamer-plugins-base1.0-dev libgnutls28-dev libxml2-dev \
    libprotobuf-dev protobuf-compiler-grpc
```

### Build

```bash
meson setup build --prefix=$HOME/.local
meson compile -C build
./build/main/dino
```

### Install (Optional)

```bash
sudo meson install -C build
sudo ldconfig
```

## Usage

1. Launch Dino
2. Open any conversation
3. Press `Ctrl++` to zoom in
4. Press `Ctrl+-` to zoom out
5. Press `Ctrl+0` to reset

Font size is saved automatically and persists across restarts.
